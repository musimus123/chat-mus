# Sistema de Chat en Línea - Electrónica Bernal

## Descripción
Sistema de chat en tiempo real desarrollado con Flask, Socket.IO y SQLite.

## Tecnologías
- Python
- Flask
- Flask-SocketIO
- SQLite
- HTML
- CSS
- JavaScript

## Funcionalidades
- Chat en tiempo real
- Historial de mensajes
- Panel de administración
- Usuarios conectados
- Estadísticas
- Diseño responsive

## Instalación

pip install -r requirements.txt

python app.py

## Acceso

http://127.0.0.1:5000
